<h3>Overview</h3>

Built an automated computational pipeline for screening antibiotic analogues using molecular dynamics simulations and structure-based analysis, dramatically increasing throughput for drug discovery research.
<div style="height: 1rem;"></div>
<div class="project-section" markdown="1">

<h3>Key Accomplishments</h3>

- Built an automated in silico screening pipeline for antibiotic analogues using molecular dynamics simulations
- Developed a Python-based workflow integrated with ChimeraX, increasing molecular simulation throughput by ~100×
- Applied AlphaFold and RDKit to perform structure-based and cheminformatics analysis on large-scale datasets
